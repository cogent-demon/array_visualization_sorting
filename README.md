# Array Visualization

A simple flutter experiment explaining how to render an array as colorful bars as the array is sorted using various sorting algorithms. The project currently supports following sorting algorithms - 

- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort
